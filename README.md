# Room Configuration Hosting

This repository hosts the room configuration JSON files for a Marauder style mapping app.  

## Purpose

- Provide a test endpoint for downloading the latest room layout JSON.  
- Supports app testing and development without embedding static files.  

## Usage

- Versioning is supported to allow updates without overwriting existing files.  
- Signature verification ensures the integrity of the downloaded file.  

> **Note:** This repository is for testing only. Production deployments should use a secure server with properly managed keys.
